Workflowy MCP

An MCP server for connecting AI assistants to your Workflowy account

How to Connect

This is an MCP (Model Context Protocol) server. To use it, configure your MCP client with:

URL: /api/mcp

Header: Authorization: Bearer YOUR_WORKFLOWY_API_KEY

Get your API key from beta.workflowy.com/api-reference

Example Configuration

For Claude Code (.claude/mcp.json):

{
  "mcpServers": {
    "workflowy": {
      "url": "https://your-app.vercel.app/api/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
